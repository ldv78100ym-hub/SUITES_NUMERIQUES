Méthodes d'étude des Suites

Suites Arithmétiques

Une suite $(u_n)$ est arithmétique si l'on passe d'un terme au suivant en ajoutant toujours le même nombre $r$, appelé la raison.

Relation de récurrence : $$u_{n+1} = u_n + r$$

Méthode : Démontrer qu'une suite est arithmétique

Pour prouver qu'une suite $(u_n)$ est arithmétique, il faut calculer la différence entre deux termes consécutifs :

  1. Exprimer $u_{n+1}$ et $u_n$ en fonction de $n$.
  2. Calculer la différence $u_{n+1} - u_n$.
  3. Simplifier l'expression. Si le résultat est une constante (un nombre réel qui ne dépend plus de $n$), alors la suite est arithmétique de raison $r$.

Méthode : Étudier les variations

Le sens de variation d'une suite arithmétique dépend uniquement du signe de sa raison $r$ :

  • Si $r > 0$, la suite est strictement croissante.
  • Si $r < 0$, la suite est strictement décroissante.
  • Si $r = 0$, la suite est constante.

Suites Géométriques

Une suite $(v_n)$ est géométrique si l'on passe d'un terme au suivant en multipliant toujours par le même nombre $q$, appelé la raison.

Relation de récurrence : $$v_{n+1} = v_n \times q$$

Méthode : Démontrer qu'une suite est géométrique

Pour prouver qu'une suite $(v_n)$ dont les termes sont non nuls est géométrique :

  1. Calculer le quotient $\frac{v_{n+1}}{v_n}$.
  2. Simplifier l'expression. Si le résultat est une constante $q$ (indépendante de $n$), alors la suite est géométrique de raison $q$.

Alternative (si un terme peut être nul) : Montrer qu'on peut écrire $v_{n+1} = q \times v_n$ où $q$ est une constante réelle.

Méthode : Étudier les variations

Pour étudier les variations d'une suite géométrique, on étudie le signe de la différence $v_{n+1} - v_n$ :

$$v_{n+1} - v_n = v_n \times q - v_n = v_n(q - 1)$$

Il faut donc étudier le signe de $v_n$ et le comparer à la position de $q$ par rapport à $1$.
